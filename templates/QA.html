This website is built for user to explore and analyze of the Electronic Theses and Dissertations (ETD) dataset from Virginia Tech, and tobacco dataset from the University of California, San Francisco. This website provides full-text search of Virginia Tech Electronic Theses and Dissertations (ETDs) and Tobacco Settlement documents.

Aside from basic searching, this website also provides personalized recommendations and data visualizations to assist users in discovering the trends and patterns of their field of interest.

Furthermore, a series of plots are provided for users to explore the data statistically. User can dynamically modify the appearance of plots to better understand the data and find information that they need.

VTechWorks:

  • The VtechWorks is free to anyone in the world with an Internet connection.
  • More beneficial and more accessible to scholars at under-resourced institutions and in developing nations.
  • It has a persistent, stable link to the item record ("handle") that won't change, which makes it more reliable for others to cite and to share on social media.

Tobacco Settlement Data:

  • The tobacco settlement data set contain 91,046,507 pages in 14,867,353 documents. It provides users with the most exhausted information on tobacco documents.
  • The query is user-friendly. Users can simply search key words or apply filters on the search page to get the information they need.

First-time users need to click the register button on the right-top of the website. Returning users need to login before using the search page.

Users firstly need to choose from one of the main topics, either ETD or Tobacco, that interest you , to start with. Users can do this by simply chick on the tab that interests you. There are several search options on right side of the homepage: keyword search box, a category browse, and filters for place name and subject.

Once an action has taken place on the search page. The user will be taken to a search results page. The text search box will still be present at the top, but additional options for filtering will be available. This is accomplished by selected terms under the facets on the screen.

  • On the top of the search page, users can find the visualization page. Once users click this button, they will be directly to the visualization page. On this page, a variety of visualizations, for instance , tables, charts, histograms, tables, and maps, of the two dataset are provided.
  • The visualizations are realized using kibana, which is built on top of Elasticsearch and is a part of the ELK stack, which consists of Elasticsearch, Logstash, and Kibana. It connects to Elasticsearch using the REST API. Users can User can modify the set-ups of the plots, which are provided on the visualization page and dynamically modify the appearance of plots to better understand the data and find information that they need.
